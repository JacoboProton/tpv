Sistema de TPV profesional para bares y restaurantes con facturaciГіn electrГіnica (Verifactu AEAT), adaptado a Canarias (IGIC 7%).

## Stack

- **Next.js 16** (App Router, Turbopack)
- **PostgreSQL Neon** (serverless)
- **React 19**, Tailwind 4, Recharts
- **Fiskaly SIGN ES** (Verifactu REST API sin SDK)
- **Stripe** (pagos con tarjeta)
- **ESC/POS** (impresiГіn tГ©rmica 80mm + WebUSB)

## Funcionalidades

| ГЃrea | DescripciГіn |
|---|---|
| **SalГіn** | Mesas/barra/para llevar/domicilio, comandas por cursos, modificadores |
| **Cocina** | Vista de tickets pendientes agrupados por curso, temporizador, urgencia |
| **TPV** | DivisiГіn de cuenta por artГ­culos, propinas, descuentos, mГєltiples mГ©todos |
| **Verifactu** | FacturaciГіn electrГіnica AEAT con cadena de huellas SHA-256, QR validable |
| **Offline** | Cache en localStorage, cola de mutaciones, sincronizaciГіn automГЎtica |
| **ImpresiГіn** | Ticket tГ©rmico vГ­a `window.print()` + WebUSB ESC/POS directo, apertura cajГіn |
| **Inventario** | Control de stock por ubicaciГіn (Bar, Cocina, AlmacГ©n), alertas stock bajo |
| **Informes** | Extracto anual, desglose mensual, top productos, exportaciГіn CSV |
| **Turnos** | Control de entrada/salida de empleados |
| **Temas** | Claro/oscuro |

## Requisitos

- Node.js 20+
- Cuenta Neon (PostgreSQL serverless)
- Cuenta Fiskaly (modo test o live)
- Cuenta Stripe (opcional, solo para pagos con tarjeta)

## ConfiguraciГіn

```bash

# Editar .env.local con tus credenciales
```
